Agente de Operaciones Inteligente de Rappi

Sistema multiagente de IA para acceso a datos operativos

GitHub

Resumen

Un sistema multiagente de IA diseñado para democratizar el acceso a datos operativos en Rappi como parte de un proceso de entrevista técnica. El sistema permite a los equipos de Estrategia, Planificación y Analítica (SP&A) obtener profundas introspecciones, visualizaciones interactivas e informes estratégicos usando lenguaje natural.


Construido usando Google ADK con Gemini 3 Flash como modelo subyacente, el sistema orquesta múltiples agentes especializados que colaboran para entender consultas, obtener datos, generar visualizaciones y producir informes accionables.


Este proyecto demuestra arquitectura de sistema multiagente lista para producción y la habilidad de bridgear sistemas de datos complejos con stakeholders no técnicos a través de interfaces intuitivas de lenguaje natural.


Características Clave

  • Colaboración Multi-Agente: Agentes especializados para comprensión de consultas, recuperación de datos, generación de visualización y redacción de informes que trabajan juntos para fulfill solicitudes de usuarios.
  • Flujo de Trabajo Dirigido por Orquestador: Un orquestador centralizado gestiona la interacción entre agentes, asegurando coordinación sin fisuras y ejecución eficiente de tareas complejas.
  • Interfaz de Lenguaje Natural: Procesamiento de lenguaje natural intuitivo para interacción sin costuras con el sistema.
  • Acceso a Datos y Visualización: Integración con fuentes de datos internas y habilidad de generar visualizaciones interactivas sobre la marcha.
  • Informes Accionables: Generación de informes estratégicos con introspecciones claras y recomendaciones basados en los datos y visualizaciones obtidos.
  • Detección de Anomalías: Pipeline de ML para detectar patrones inusuales en los datos y alertar a los stakeholders relevantes cuando las métricas se deterioran.
