Panel de Análisis de Salarios Docentes

Visualización interactiva de datos salariales docentes en Argentina

GitHub Demo

Resumen

Una aplicación web interactiva construida con Dash para visualizar y analizar datos de salarios docentes (categoría MG10) en las 24 jurisdicciones de Argentina.


Cuenta con ajuste dinámico por inflación, permitiendo la comparación en tiempo real del poder adquisitivo entre provincias y períodos. El benchmarking provincial permite entender la compensación relativa e identificar disparidades.


Este proyecto demuestra experiencia en la creación de aplicaciones de datos que traducen información fiscal compleja en insights accionables—una habilidad perfeccionada durante mi etapa como asesor de políticas públicas.


Características Clave

  • Ajuste por Inflación: Los salarios se ajustan mediante el Índice de Precios al Consumidor (IPC) del INDEC, permitiendo comparaciones precisas de salarios reales en el tiempo.
  • Benchmarking Provincial: Comparación de salarios entre las 24 jurisdicciones, identificando tendencias y brechas en la compensación docente.
  • Visualizaciones Interactivas: Incluye gráficos de líneas para comparación histórica frente a líneas de pobreza y gráficos de barras para comparar jurisdicciones.
  • Analítica Avanzada y ML: Incorpora Machine Learning para agrupar jurisdicciones similares mediante KShape y detección de anomalías con IsolationForest.
